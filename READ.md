# SmartHire AI

## Overview

SmartHire AI is a multi-agent resume screening and candidate evaluation system that helps recruiters automate resume analysis.

## Features

* Resume Upload
* Resume Parsing
* Skill Extraction
* Resume Scoring
* Candidate Evaluation
* AI-Powered Resume Analysis
* Interview Question Generation

## Technologies Used

* Python
* Streamlit
* PDFPlumber
* Gemini API

## Workflow

1. Upload Resume
2. Extract Resume Text
3. Detect Skills
4. Score Resume
5. Evaluate Candidate
6. Generate AI Feedback

## Future Enhancements

* Multi-resume ranking
* Job description upload
* Recruiter dashboard
* Resume recommendation system
